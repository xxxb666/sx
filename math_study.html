第二章：有理数及其运算

▼

📌 数轴

📌 相反数与绝对值

  • 有理数的分类：
    • 整数：正整数、0、负整数
    • 分数：正分数、负分数
  • 数轴三要素：原点、正方向、单位长度
  • 相反数：只有符号不同的两个数，a 的相反数是 -a，0 的相反数是 0
  • 绝对值：
    • 几何意义：数轴上表示数的点到原点的距离
    • 代数意义：|a| = a (a>0), |a| = 0 (a=0), |a| = -a (a<0)
    • 性质：|a| ≥ 0，|a| = |-a|
  • 有理数加法法则：
    • 同号两数相加，取相同的符号，并把绝对值相加
    • 异号两数相加，取绝对值较大的加数的符号，并用较大的绝对值减去较小的绝对值
    • 一个数同 0 相加，仍得这个数
  • 有理数减法法则：减去一个数，等于加上这个数的相反数。a - b = a + (-b)
  • 有理数乘法法则：
    • 两数相乘，同号得正，异号得负，并把绝对值相乘
    • 任何数与 0 相乘，都得 0
  • 有理数除法法则：
    • 除以一个不等于 0 的数，等于乘这个数的倒数。a ÷ b = a × (1/b) (b≠0)
    • 两数相除，同号得正，异号得负，并把绝对值相除
  • 乘方：求 n 个相同因数的积的运算，记作 aⁿ，a 是底数，n 是指数
  • 科学记数法：把一个大于 10 的数表示成 a×10ⁿ 的形式（1≤|a|<10，n 是正整数）
  • 有理数混合运算顺序：先算乘方，再算乘除，最后算加减；如果有括号，先算括号里面的

第三章：整式及其加减

▼

📌 单项式与多项式

📌 合并同类项

  • 代数式：用运算符号把数和表示数的字母连接而成的式子
  • 代数式的值：用数值代替代数式中的字母，按照代数式指明的运算进行计算
  • 单项式：数与字母的积，单独的一个数或一个字母也是单项式
  • 单项式的系数：单项式中的数字因数
  • 单项式的次数：单项式中所有字母的指数和
  • 多项式：几个单项式的和
  • 多项式的项：多项式中的每个单项式
  • 多项式的次数：多项式里次数最高项的次数
  • 整式：单项式和多项式统称为整式
  • 同类项：所含字母相同，并且相同字母的指数也相同的项
  • 合并同类项法则：系数相加，字母和字母的指数不变
  • 去括号法则：
    • 括号前面是"+"号，把括号和它前面的"+"号去掉，括号里各项都不改变符号
    • 括号前面是"-"号，把括号和它前面的"-"号去掉，括号里各项都改变符号
  • 整式的加减：几个整式相加减，有括号先去括号，然后合并同类项

第四章：基本平面图形

▼

📌 线段、射线、直线

📌 角的构成

📌 角的分类

  • 线段：有两个端点，可以度量长度
  • 射线：有一个端点，向一方无限延伸，不可度量
  • 直线：没有端点，向两方无限延伸，不可度量
  • 线段的基本事实：两点之间，线段最短
  • 两点确定一条直线
  • 线段的中点：把一条线段分成两条相等线段的点
  • 角：有公共端点的两条射线组成的图形，公共端点是顶点
  • 角的表示：∠AOB、∠O、∠1、∠α
  • 角的度量：1°=60′，1′=60″
  • 角的平分线：从一个角的顶点出发，把这个角分成两个相等的角的射线
  • 余角和补角：
    • 余角：如果两个角的和等于 90°，那么这两个角互为余角
    • 补角：如果两个角的和等于 180°，那么这两个角互为补角
    • 性质：同角（或等角）的余角相等；同角（或等角）的补角相等
  • 多边形：由一些线段首尾顺次连接组成的封闭图形
  • 圆：到定点的距离等于定长的所有点组成的图形

第五章：一元一次方程

▼

📌 方程示例

📌 解方程步骤

  • 方程：含有未知数的等式
  • 一元一次方程：只含有一个未知数，并且未知数的次数是 1 的整式方程
  • 方程的解：使方程左右两边相等的未知数的值
  • 等式的基本性质：
    • 性质 1：等式两边同时加上（或减去）同一个代数式，所得结果仍是等式
    • 性质 2：等式两边同时乘同一个数（或除以同一个不为 0 的数），所得结果仍是等式
  • 解一元一次方程的一般步骤：
    • 去分母（方程两边都乘各分母的最小公倍数）
    • 去括号（先去小括号，再去中括号，最后去大括号）
    • 移项（把含有未知数的项移到方程的一边，常数项移到另一边）
    • 合并同类项
    • 系数化为 1（方程两边都除以未知数的系数）
  • 列方程解应用题的一般步骤：
    • 审：审题，分析已知量和未知量
    • 设：设未知数
    • 列：列出方程
    • 解：解方程
    • 答：检验并写出答案
  • 常见应用题类型：
    • 行程问题：路程 = 速度 × 时间
    • 工程问题：工作总量 = 工作效率 × 工作时间
    • 利润问题：利润 = 售价 - 进价，利润率 = 利润÷进价×100%
    • 储蓄问题：利息 = 本金 × 利率 × 期数

第六章：数据的收集与整理

📌 统计图类型

📌 数据收集方法

  • 数据的收集方法：问卷调查、访谈、查阅资料、实验、观察等
  • 普查：为某一特定目的而对所有考察对象进行的全面调查
  • 抽样调查：从总体中抽取部分个体进行调查
  • 总体、个体、样本：
    • 总体：所要考察对象的全体
    • 个体：组成总体的每一个考察对象
    • 样本：从总体中抽取的一部分个体
    • 样本容量：样本中个体的数目
  • 统计图：
    • 条形统计图：能清楚地表示出每个项目的具体数目
    • 折线统计图：能清楚地反映事物的变化情况
    • 扇形统计图：能清楚地表示出各部分在总体中所占的百分比
  • 扇形统计图：
    • 圆心角 = 360° × 百分比
    • 所有扇形圆心角的和 = 360°
    • 所有百分比的和 = 1

第二章：有理数及其运算

题目 1：有理数加法

计算：(-3) + (-5) - (-8) × 2

解答

解：原式 = -3 + (-5) - (-16)

        = -8 + 16

        = 8

题目 2：绝对值计算

计算：|-5| + |3| - |-2|

解答

解：原式 = 5 + 3 - 2

        = 6

题目 3：有理数混合运算

计算：-2² + (-3)² × 2 - 4 ÷ (-2)

解答

解：原式 = -4 + 9 × 2 - (-2)

        = -4 + 18 + 2

        = 16

题目 4：相反数与绝对值

已知|a| = 3，|b| = 5，且 a < b，求 a + b 的值。

解答

解：∵ |a| = 3，|b| = 5

∴ a = ±3，b = ±5

又∵ a < b

∴ b = 5，a = 3 或 a = -3

当 a = 3 时，a + b = 3 + 5 = 8

当 a = -3 时，a + b = -3 + 5 = 2

综上所述，a + b = 8 或 2

题目 5：科学记数法

解答

答案：3.5 × 10⁶

解析：3500000 = 3.5 × 1000000 = 3.5 × 10⁶

第三章：整式及其加减

题目 1：合并同类项

化简：3x²y - 2xy² + 5x²y + 4xy²

解答

解：原式 = (3x²y + 5x²y) + (-2xy² + 4xy²)

        = 8x²y + 2xy²

题目 2：去括号

化简：2(a + b) - 3(a - b)

解答

解：原式 = 2a + 2b - 3a + 3b

        = -a + 5b

题目 3：整式的加减

已知 A = 3x² - 2x + 1，B = x² + 4x - 5，求 A - 2B。

解答

解：A - 2B = (3x² - 2x + 1) - 2(x² + 4x - 5)

                = 3x² - 2x + 1 - 2x² - 8x + 10

                = x² - 10x + 11

题目 4：单项式的系数和次数

单项式 -2³xy²的系数是______，次数是______。

解答

答案：系数是 -8，次数是 3

解析：-2³xy² = -8xy²，系数是 -8，次数是 1+2=3

题目 5：化简求值

先化简，再求值：2(x²y + xy²) - 3(x²y - 1) - 2xy² - 4，其中 x = 1，y = -1。

解答

解：原式 = 2x²y + 2xy² - 3x²y + 3 - 2xy² - 4

        = -x²y - 1

当 x = 1，y = -1 时，

原式 = -1² × (-1) - 1 = 1 - 1 = 0

第四章：基本平面图形

题目 1：角度计算

已知∠AOB = 60°，OC 平分∠AOB，求∠AOC。

解答

解：∵ OC 平分∠AOB

∴ ∠AOC = 1/2 ∠AOB = 1/2 × 60° = 30°

题目 2：角度换算

计算：45°30′ + 23°45′

解答

解：45°30′ + 23°45′ = 68°75′

        = 69°15′

题目 3：余角和补角

一个角的补角比它的余角大______度。

解答

答案：90

解析：设这个角为 x°，则它的补角是 (180-x)°，余角是 (90-x)°

(180-x) - (90-x) = 180 - x - 90 + x = 90°

题目 4：线段的中点

已知线段 AB = 10cm，点 C 是 AB 的中点，点 D 是 BC 的中点，求 AD 的长。

解答

解：∵ C 是 AB 的中点，AB = 10cm

∴ AC = BC = 5cm

又∵ D 是 BC 的中点

∴ CD = 2.5cm

∴ AD = AC + CD = 5 + 2.5 = 7.5cm

题目 5：角的计算

如图，O 是直线 AB 上一点，∠AOC = 50°，OD 平分∠BOC，求∠COD 的度数。

解答

解：∵ O 是直线 AB 上一点

∴ ∠AOB = 180°

∴ ∠BOC = 180° - ∠AOC = 180° - 50° = 130°

又∵ OD 平分∠BOC

∴ ∠COD = 1/2 ∠BOC = 1/2 × 130° = 65°

第五章：一元一次方程

题目 1：解方程

解方程：3(x - 2) + 1 = 2x - 5

解答

解：去括号：3x - 6 + 1 = 2x - 5

移项：3x - 2x = -5 + 6 - 1

合并：x = 0

题目 2：解方程

解方程：2x + 3 = 5x - 9

解答

解：移项：2x - 5x = -9 - 3

合并：-3x = -12

系数化 1：x = 4

题目 3：解带分母的方程

解方程：(x-1)/2 - (2x+1)/3 = 1

解答

解：去分母，方程两边都乘 6：

3(x-1) - 2(2x+1) = 6

去括号：3x - 3 - 4x - 2 = 6

移项：3x - 4x = 6 + 3 + 2

合并：-x = 11

系数化 1：x = -11

题目 4：方程的应用（行程问题）

甲、乙两地相距 120km，A 车从甲地出发，速度为 60km/h；B 车从乙地出发，速度为 40km/h。两车同时出发，相向而行，几小时后相遇？

解答

解：设 x 小时后两车相遇。

根据题意，得：60x + 40x = 120

合并：100x = 120

解得：x = 1.2

答：1.2 小时后两车相遇。

题目 5：方程的应用（利润问题）

某商品进价为 100 元，标价为 150 元，现打折销售，利润率为 20%，问打了几折？

解答

解：设打了 x 折。

根据题意，得：150 × 0.1x - 100 = 100 × 20%

15x - 100 = 20

15x = 120

x = 8

答：打了 8 折。
